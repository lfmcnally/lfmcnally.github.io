Second Declension -r Nouns

vir, puer, and liber - the special cases

The Special Cases: -r Endings

However, a few 2nd declension masculine nouns have a nominative singular ending in -r instead of -us. The most important ones are:

vir
man
Pattern: Keeps the same stem throughout
Nom: vir | Acc: virum | Gen: viri
puer
boy
Pattern: Keeps the -e- in all cases
Nom: puer | Acc: puerum | Gen: pueri
liber
book
Pattern: Drops the -e- from accusative onwards
Nom: liber | Acc: librum | Gen: libri
ager
field
Pattern: Like liber - drops the -e-
Nom: ager | Acc: agrum | Gen: agri
Key Point: These nouns decline as if they had a nominative singular ending -us which has disappeared. All their other endings are exactly the same as dominus!

Recognising the Patterns

There are three distinct patterns to learn. Understanding which pattern a noun follows is crucial for getting the other cases right:

Type 1: Like vir
Simple -r ending
Pattern: Just add normal endings to the stem
Example: vir → virum, viri, viro
No complications - straightforward!
Type 2: Like puer
Keeps the -e-
Pattern: The -e- stays in all cases
Example: puer → puerum, pueri, puero
Helpful because it sounds right when spoken!
Type 3: Like liber
Drops the -e-
Pattern: The -e- disappears from accusative onwards
Example: liber → librum, libri, libro
Also: ager → agrum, agri, agro
Watch out for this one - easy to get wrong!

Practice Examples

Let's see these nouns in action! Click each sentence to reveal the translation and notice which pattern each noun follows:

-r Nouns in Context
vir puerum vocat.
Breakdown: vir (man - nom sg), puerum (boy - acc sg), vocat (calls)
Translation: The man calls the boy.
pueri libros legunt.
Breakdown: pueri (boys - nom pl), libros (books - acc pl), legunt (read)
Translation: The boys read the books.
domini in agris laborant.
Breakdown: domini (masters - nom pl), in agris (in the fields - prep + abl pl), laborant (work)
Translation: The masters work in the fields.
librum viri legimus.
Breakdown: librum (book - acc sg), viri (of the man - gen sg), legimus (we read)
Translation: We read the man's book.
Pattern Spotting: Notice puerum (keeps the -e-), libros (drops the -e-), agris (also drops the -e-), and viri (simple -r pattern). Each follows its type consistently!

Memory Tips

Pronunciation Guide
How to remember the patterns
puer: "POO-er-um" - the -e- makes it sound natural
liber: "LEE-brum" - dropping the -e- sounds better
ager: "AH-grum" - same as liber pattern
Trust your ear - Latin flows better this way!
Dictionary Clue
How to tell which pattern
In vocabulary lists: The genitive tells you!
puer, pueri: Keeps the -e- (you see "pueri")
liber, libri: Drops the -e- (you see "libri" not "liberi")
Always check the genitive form to be sure!
