Thank you for checking out Classicalia! I'm Lawrence and I teach Latin and Classics in Bristol.

This Look, Cover, Write, Check tool is a classic vocabulary learning method. Study the words, then test yourself from memory. Each round builds on the last, so you can see which words need more attention.

Chunking: Start with a small group of words and unlock more as you get comfortable. This stops you feeling overwhelmed!
