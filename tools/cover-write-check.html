# Latin English Chapter

How To Use

  • 1. Enter vocabulary pairs in the table
  • 2. Set column headers (e.g. Latin / English)
  • 3. Choose how many attempt columns to include
  • 4. Optionally add a chapter/section column
  • 5. Hit Print to generate a clean worksheet

How It Works

Students look at the word, cover it, write it from memory, then check their answer.

Multiple attempt columns let students practise the same words several times on one sheet.

The chapter column is useful for organising vocabulary by textbook section.
